# K Ramya — Portfolio

**Live site: [kraya-23.github.io](https://kraya-23.github.io)**

My personal portfolio — designed and built by hand, every pixel and every wiggle. A paper-and-marker style site with doodles, stickers, ink trails and scroll animations, showcasing my web design experiments.

| Project | Description |
|---|---|

## Built with

- Pure HTML, CSS and vanilla JavaScript — no frameworks, no build step
- Hand-rolled animations: scroll reveals, marquees, cursor ink trail, click stamps, 3D card tilt
- Fully responsive, keyboard-friendly, `prefers-reduced-motion` respected
